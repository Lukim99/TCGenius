# 전투력 안내

전투력은 캐릭터가 얼마나 강한지를 한눈에 보여주기 위한 점수입니다.
단순히 공격력만 보는 것이 아니라, 생존력, 스킬 사용 편의성, 골드/경험치/아이템 획득 능력까지 함께 반영합니다.

```text
전투력 = 공격 점수 + 생존 점수 + 편의/보상 점수
```

## 1. 공격 점수

공격 점수는 몬스터에게 얼마나 큰 피해를 줄 수 있는지를 나타냅니다.

공격 점수를 올리는 대표 스탯은 다음과 같습니다.

- **공격력**: 가장 기본이 되는 공격 점수입니다.
- **최종 공격력%**: 공격력을 비율로 올려 공격 점수를 크게 높입니다.
- **일반 공격 피해%**: 일반 사냥 공격 피해를 올립니다.
- **스킬 공격 피해%**: 스킬 피해를 올립니다. 전투력에는 일부 비율로 반영됩니다.
- **일반 몬스터에게 주는 피해 증가%**: 일반 몬스터 상대 화력이 증가합니다.
- **엘리트 몬스터 대상 추가 피해%**: 엘리트 몬스터 상대 화력이 증가합니다.
- **방어 관통력**: 몬스터 방어력을 무시하는 능력입니다.
- **방어력 관통%**: 몬스터 방어력을 비율로 무시하는 능력입니다.
- **치명타 확률**: 치명타가 터질 확률입니다.
- **치명타 피해량**: 치명타가 터졌을 때 피해량을 올립니다.
- **연격 확률**: 추가 공격이 이어질 확률입니다.
- **추가 공격 횟수**: 연격이 이어질 수 있는 최대 횟수를 늘립니다.
- **공격 시 10/100/1000 추가 피해 확률**: 공격할 때 추가 고정 피해가 붙을 확률입니다.
- **스킬 사용 시 추가 고정 피해**: 스킬 공격에 고정 피해를 더합니다.
- **방어 무시 피해 확률**: 특정 장비 효과로 발생하며, 전투력에 공격 보정으로 반영됩니다.

쉽게 말하면, 공격 점수는 아래 요소들이 곱해져 커집니다.

```text
공격 점수 = 공격력 기반 점수
× 피해 증가 효과
× 치명타 기대값
× 연격 기대값
× 방어 관통 효과
× 추가 피해 효과
```

## 2. 생존 점수

생존 점수는 사냥 중 얼마나 오래 버틸 수 있는지를 나타냅니다.

생존 점수를 올리는 대표 스탯은 다음과 같습니다.

- **체력**: 생존 점수의 기본이 됩니다.
- **최종 체력%**: 체력을 비율로 올립니다.
- **방어력**: 받는 피해를 줄여 생존 점수를 올립니다.
- **최종 방어력%**: 방어력을 비율로 올립니다.
- **회피 확률**: 공격을 피할 수 있어 생존 점수가 크게 오릅니다.
- **받는 피해 증가%**: 양수면 생존 점수가 낮아지고, 음수면 받는 피해 감소로 보고 생존 점수가 올라갑니다.
- **회복 효율%**: HP/MP 회복량이 증가하므로 생존 점수에 일부 반영됩니다.
- **치명타 피해 감소율**: 적의 치명타 피해를 줄여 생존 점수를 올립니다.
- **사냥 시 HP 소모량 감소**: 카드 슬롯 효과로, 받는 피해 감소처럼 생존 점수에 반영됩니다.
- **적 처치 시 잃은 HP 회복 확률**: 카드 슬롯 효과로, 사냥 유지력을 올립니다.

쉽게 말하면, 생존 점수는 아래 요소로 정해집니다.

```text
생존 점수 = 체력과 방어력으로 만든 기본 생존력
× 회피 효과
× 피해 감소 효과
× 회복 효과
× 치명타 피해 감소 효과
```

## 3. 편의/보상 점수

편의/보상 점수는 직접 피해량은 아니지만 사냥 효율에 도움을 주는 능력입니다.

편의/보상 점수를 올리는 대표 스탯은 다음과 같습니다.

- **MP**: 스킬을 더 안정적으로 사용할 수 있어 점수에 반영됩니다.
- **최종 MP%**: MP를 비율로 올립니다.
- **MP 소모량 감소**: 스킬 MP 소모가 줄어들면 점수가 올라갑니다.
- **스킬 쿨타임 감소**: 스킬을 더 자주 쓸 수 있어 점수가 올라갑니다.
- **골드 획득량%**: 골드 파밍 효율이 올라갑니다.
- **경험치 획득량%**: 성장 효율이 올라갑니다.
- **처치 당 골드**: 몬스터 처치 시 얻는 골드가 고정 수치만큼 늘어나며, 전투력에도 반영됩니다.
- **물약 효율%**: 회복 아이템 효율이 좋아져 보조 점수에 반영됩니다.
- **아이템 획득 확률%**: 아이템 파밍 효율이 올라갑니다.

쉽게 말하면, 편의/보상 점수는 아래 요소로 정해집니다.

```text
편의/보상 점수 = MP와 스킬 사용 편의성
+ 골드/경험치/아이템 획득 효율
+ 물약 효율
```

## 4. 카드 슬롯 효과 반영

캐릭터 카드 슬롯 효과도 전투력에 반영됩니다.

| 효과 | 전투력 반영 |
| --- | --- |
| 경험치 획득 증가량 | 편의/보상 점수 증가 |
| 사냥 시 HP 소모량 감소 | 생존 점수 증가 |
| 적 처치 시 잃은 HP 회복 확률 | 생존 점수 증가 |
| 치명타 확률 증가 | 공격 점수 증가 |
| 사냥 시 MP 소모량 감소 | 편의/보상 점수 증가 |
| 일반 몬스터에게 주는 피해 증가 | 공격 점수 증가 |
| 치명타 피해량 증가 | 공격 점수 증가 |
| 골드 획득 증가량 | 편의/보상 점수 증가 |
| 아이템 드랍 확률 | 편의/보상 점수 증가 |
| 방어력 관통 | 공격 점수 증가 |
| 일반 공격 피해량 증가 | 공격 점수 증가 |

## 6. 주의할 점

- 전투력은 실제 전투 결과를 완벽히 그대로 보여주는 값은 아닙니다.
- 치명타, 연격, 추가 피해처럼 확률이 있는 스탯은 기대값으로 계산됩니다.
- 골드, 경험치, 아이템 획득 관련 스탯도 사냥 효율이므로 전투력에 포함됩니다.
- 특정 상황에서만 강한 스탯도 전투력에 일부 반영됩니다.
  - 예: 엘리트 몬스터 대상 추가 피해
  - 예: 일반 몬스터에게 주는 피해 증가
- `받는 피해 증가`처럼 이름상 페널티인 스탯은 양수일 때 전투력을 낮춥니다.
- `MP 소모량`, `스킬 쿨타임`은 감소 효과일 때 전투력이 올라갑니다.

## 7. 간단한 해석법

전투력이 올랐다는 것은 보통 아래 중 하나 이상이 좋아졌다는 뜻입니다.

- **더 강하게 때린다**
- **더 오래 버틴다**
- **스킬을 더 편하게 쓴다**
- **사냥 보상을 더 많이 얻는다**

따라서 순수 전투만 중요하면 공격/생존 관련 스탯을 보면 되고,
파밍 효율까지 중요하면 골드, 경험치, 아이템 획득 관련 스탯도 함께 보면 됩니다.
