# RPGenius 명령어 가이드

> 접두사: `/RPGenius` 또는 `/rpg`
> 예시: `/RPGenius 등록 홍길동` = `/rpg 등록 홍길동`
> `/rpg 도움말` 으로 인게임에서도 확인 가능

---

## 1. 계정 & 캐릭터

| 명령어 | 설명 |
|--------|------|
| `/rpg 등록 [닉네임]` | 신규 계정(Owner) 등록 |
| `/rpg 캐릭터생성 [이름] [직업]` | 새 캐릭터 생성 (최대 5개) |
| `/rpg 캐릭터목록` | 보유 캐릭터 목록 확인 |
| `/rpg 캐릭터선택 [번호]` | 활성 캐릭터 변경 |
| `/rpg 정보` | 현재 캐릭터 상세 정보 |

**직업 목록:** `먼마`, `성준호`, `빵귤`, `호르아크티`, `건마`

---

## 2. 전투 & 던전

| 명령어 | 설명 |
|--------|------|
| `/rpg 탐험` | 입장 가능한 던전 목록 표시 |
| `/rpg 입장 [번호]` | 던전에 입장하여 전투 시작 (피로도 소모) |
| `/rpg 공격` | 일반 공격 |
| `/rpg 스킬 [스킬명]` | 스킬 사용 (전투 중) |
| `/rpg 아이템 [아이템명]` | 아이템 사용 (전투 중) |
| `/rpg 도망` | 전투에서 도주 시도 |

- 던전은 페이즈 시스템으로 구성 (1~3페이즈)
- 피로도: 기본 156, 매일 자정 리셋
- 보스 몬스터는 무력게이지(스태거) 시스템 적용

---

## 3. 스탯 & 스킬

| 명령어 | 설명 |
|--------|------|
| `/rpg 스킬` | 보유 스킬 목록 및 쿨타임 확인 |
| `/rpg 스킬레벨업 [스킬명]` | SP를 소모하여 스킬 레벨업 |
| `/rpg 스탯투자 [스탯명] [수치]` | 스탯 포인트 투자 |

**스탯:** `힘(power)`, `속도(speed)`, `지능(int)`, `행운(luck)`

---

## 4. 인벤토리 & 장비

| 명령어 | 설명 |
|--------|------|
| `/rpg 인벤토리` | 인벤토리 확인 (장비 + 소모품) |
| `/rpg 장착 [장비번호]` | 인벤토리 장비를 장착 (기존 장비는 인벤토리로 이동) |
| `/rpg 해제 [슬롯명]` | 장착 중인 장비 해제 (무기/투구/상의/하의/신발/장갑/목걸이/반지/팔찌) |
| `/rpg 장비정보 [장비번호]` | 장비 상세 정보 (스탯, 강화, 증폭, 귀속) |
| `/rpg 장비목록` | 현재 장착 중인 장비 확인 |
| `/rpg 사용 [아이템명]` | 소모품 아이템 사용 (비전투 중) |

**장비 슬롯:** `무기`, `투구`, `상의`, `하의`, `신발`, `장갑`, `목걸이`, `반지`, `팔찌`

**장비 등급:** 일반 → 레어 → 레전더리 → 에픽

---

## 5. 강화 & 증폭

| 명령어 | 설명 |
|--------|------|
| `/rpg 강화 [장비번호]` | 무기 강화 (강화석 + 골드 소모, 최대 +16) |
| `/rpg 증폭 [장비번호] [스탯]` | 방어구 증폭 (증폭서 + 골드 소모, 최대 +16) |

- 강화 실패 시 단계 하락 가능 (+11 이상 파괴 가능)
- 증폭은 차원 스탯(power/speed/int/luck) 선택 필요
- 최초 증폭 시 스탯 지정 필수, 이후 생략 가능

## 7. 제작

| 명령어 | 설명 |
|--------|------|
| `/rpg 제작목록` | 제작 가능한 레시피 목록 (재료 표시) |

- 장비, 소모품, 강화 재료 등 다양한 레시피
- 재료 + 골드 소모

## 8. 낚시

| 명령어 | 설명 |
|--------|------|
| `/rpg 낚시 [낚싯대] [떡밥]` | 낚시 시작 (떡밥 1개 소모) |
| `/rpg 낚시 정보` | 보유 낚싯대, 떡밥 확인 |

- 물고기, 재료, 봉인된 자물쇠 등 획득 가능
- 낚싯대 등급에 따라 대기 시간 및 확률 변동

## 13. 경매장 & 거래소 & 우편

### 경매장 (골드 거래)
| 명령어 | 설명 |
|--------|------|
| `/rpg 경매장` 또는 `/rpg 경매장 [페이지]` | 매물 목록 |
| `/rpg 경매장 검색 [키워드]` | 매물 검색 |
| `/rpg 경매장 등록 [아이템명] [수량] [가격]` | 아이템 판매 등록 |
| `/rpg 경매장 구매 [매물번호]` | 매물 구매 (골드 소모) |
| `/rpg 경매장 취소 [매물번호]` | 등록 매물 취소 |

- 판매 수수료 5%
- 귀속 장비는 등록 불가

### 거래소 (가넷 거래)
| 명령어 | 설명 |
|--------|------|
| `/rpg 거래소` 또는 `/rpg 거래소 [페이지]` | 매물 목록 |
| `/rpg 거래소 등록 [아이템명] [수량] [가격]` | 거래소에 등록 (골드 상자만 가능) |
| `/rpg 거래소 구매 [매물번호]` | 매물 구매 (가넷 소모) |

### 우편
| 명령어 | 설명 |
|--------|------|
| `/rpg 우편함` | 받은 우편 목록 |
| `/rpg 우편 수령 [우편번호]` | 개별 우편 수령 |
| `/rpg 우편 전체수령` | 모든 우편 수령 |
| `/rpg 우편 보내기 [닉네임] 아이템 [아이템명] [수량]` | 아이템 우편 발송 |
| `/rpg 우편 보내기 [닉네임] 가넷 [수량]` | 가넷 우편 발송 |

## 14. 봉인된 자물쇠 (가챠)

| 명령어 | 설명 |
|--------|------|
| `/rpg 자물쇠열기` | 봉인된 자물쇠 개봉 (지니어스의 열쇠 + 봉인된 자물쇠 각 1개 소모) |

- 1회 오픈 시 2개 아이템 획득
- 강화석, 증폭서, 장비 업그레이드 티켓 등 120+종 드랍

---

## 15. 시즌패스

| 명령어 | 설명 |
|--------|------|
| `/rpg 시즌패스` | 현재 패스 레벨, 경험치, 미수령 보상 확인 |
| `/rpg 시즌패스 보상` | 미수령 보상 일괄 수령 |

- 던전 클리어, 강화, 증폭 등으로 패스 경험치 획득
- 30레벨 이후 반복 보상 (강화석 5개)
- 플러스 패스: 추가 보상 지급

**경험치 획득:**
| 행동 | 획득량 |
|------|--------|
| 던전 클리어 | 30 |
| 강화 시도 | 10 |
| 증폭 시도 | 15 |

---

## 16. 화폐 & 피로도

| 명령어 | 설명 |
|--------|------|
| `/rpg 지갑` | 보유 골드/가넷 확인 |
| `/rpg 피로도` | 현재 피로도 확인 (게이지 바 표시) |
| `/rpg 도움말` | 전체 명령어 카테고리 요약 |

- **골드**: 던전 보상, 아이템 판매 등으로 획득. 강화/제작/경매장에서 사용
- **가넷**: 캐시 화폐. 거래소/시즌패스/상점에서 사용
- **피로도**: 최대 156, 매일 00:00 리셋, 던전 입장 시 소모
